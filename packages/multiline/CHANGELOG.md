# @cli-prompts/multiline

## [1.0.0] - 2025-11-16

### Added

- **Initial Release:**
  - True multiline editing with an intuitive, native-like textarea experience in the terminal.
  - Intuitive key-bindings: `Enter` to submit, `Shift+Enter` for newlines.
  - Advanced editing features, including word-wise navigation (`Ctrl+Left/Right`) and deletion (`Alt+Backspace`, `Ctrl+W`).
  - Robust, flicker-free rendering engine for a smooth user experience.
  - Support for rich features like placeholders, default values, validation, and max length.
  - Optional animated spinners for a dynamic feel during async operations.
  - Deeply customizable styling for prefixes, colors, and more.
  - Lightweight with zero runtime dependencies.
